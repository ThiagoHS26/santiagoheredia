Geomaps Security

Es una aplicación que utiliza datos geoespaciales para identificar patrones delictivos y realizar análisis en una zona. La aplicación está construida con JavaScript y utiliza MongoDB como base de datos. Está aplicación también agrupa incidentes similares utilizando el algoritmo DBSCAN y muestra estos puntos en un mapa iteractivo de Leaflet. El objetivo de esta aplicación es permitir al usuario registrar un evento criminalístico con el fin de crear patrones y áreas de riesgo para tomar decisiones de seguridad más informadas. Puedes ingresar a la aplicación con una cuenta de Google.

Visita mi repositorio en GitHub y descargalo. Encontrarás un archivo con las instrucciones necesarias para que lo apliques a tu interés
