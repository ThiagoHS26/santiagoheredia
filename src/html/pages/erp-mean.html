Genesys ERP

Es una aplicación web para llevar un control interno de procesos administrativos de un almacén. En principio, esta aplicación fue desarrollada como parte del proyecto final de carrera, cuyo objetivo era llevar un control de existencias de un taller mecánico, puedes obtener mayor información presionando aquí. La primera versión de la aplicación estaba desarrollada con Mean Stack, abreviación de: Mongo, Express, Angular, Node. Sin embargo, el proyecto fue escalando hasta la versión que se presenta ahora, la cual incluye una base de datos MySQL, para un control más estandarizado, una arquitectura modular y desacoplada, y Docker. A futuro se ha planificado añadir módulos de notificación, mensajería y microservicios.

Visita mi repositorio en GitHub y descarga el FRONTEND en Angular y el BACKEND en Node.
